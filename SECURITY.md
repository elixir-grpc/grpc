# Security Policy

## Supported Versions

Use this section to tell people about which versions of your project are
currently being supported with security updates.

| Version      | Supported          |
| -------------| ------------------ |
| 1.0.0-rc.1   | :white_check_mark: |
| >= 0.11.0    | :white_check_mark: |
| < 0.11.0     | :x:                |

## Reporting a Vulnerability

Use the private advisory features in GitHub to report vulnerabilities or security issues.
Make sure to include library versions for `:grpc`, `:grpc_server`, `:grpc_core` along with Elixir and Erlang versions.
If using git dependencies in `mix.exs`, please include the relevant mix.lock lines if possible.

Finally, include all details related to the vulnerability such as code affected and minimal reproductions if any.
